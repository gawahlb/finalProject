Welcome to the Pokemon Stat Comparison App!

Steps for using this application:

  • Navigate to the Compare page.
  • Select stats you would like to view.
  • Select two Pokemon that you would like to compare.
  • The selected Pokemon data will automatically populate!

If you would like to save a search, you can click the "Save Search" button below the search. You can access your saved views in the Saved page.

Note: You can only have three saved views at a time.
